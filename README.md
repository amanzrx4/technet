## Description

Tech Net is a social networking platform for tech enthusiasts. Create profiles , make an impressive portfolio and connect with other techies! 

## Tech Stack

React
Redux ( for state management ) 
Node.js
Express 
MongoDB

##  Features

User authentication

User profiles

Posts, Like and Comment
